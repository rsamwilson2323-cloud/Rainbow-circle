# 🌈 Rainbow Circle

An interactive HTML5 Canvas animation that creates glowing rainbow-colored orbit circles with beautiful light trails and dynamic color transitions. Designed for both desktop and mobile devices, users can generate expanding neon rings through mouse, touch, and keyboard interactions.

## ✨ Features

* 🌈 Expanding rainbow orbit circles
* ✨ Dynamic color-changing glow effects
* 🖱️ Mouse click and drag support
* 📱 Mobile touch interaction
* ⌨️ Keyboard-controlled movement effects
* 🎨 Smooth neon light trails
* ⚡ Real-time Canvas rendering
* 🖥️ Fullscreen responsive design
* 🚀 Lightweight and dependency-free

## 🎮 Controls

### Desktop

* Click to create glowing rainbow circles.
* Hold and drag to continuously generate orbit effects.
* Use Arrow Keys to push active circles in different directions.

### Mobile

* Tap anywhere to generate rainbow rings.
* Touch and drag to create continuous glowing orbit animations.
